## Project Description
A simple web app built for the Eneba internship task.

## Tech Stack
- React (Vite) for frontend
- Node.js (Express) for backend
- PostgreSQL for database

## Getting Started

### 1. Database
Create a PostgreSQL database.
Update database credentials in your backend configuration.
Seed the database with initial data:
```
node seed.js
```

### 2. Backend
```
cd backend
npm install
node index
```
Backend runs on http://localhost:3001

### 3. Frontend
```
cd frontend
npm intall
npm run dev
```
Frontend run on http://localhost:5173


Now you can access the website through http://localhost:5173
